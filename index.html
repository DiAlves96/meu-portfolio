Calculadora IMC

Calculadora IMC

Calculadora IMC

Este projeto é uma calculadora de IMC (Índice de Massa Corporal) desenvolvida para permitir que o usuário insira seu peso e altura e obtenha rapidamente seu IMC, além de uma classificação sobre sua condição corporal (como "peso normal", "sobrepeso", etc.).

Principais funcionalidades:
- Cálculo automático do IMC com base no peso e altura informados.
- Classificação do resultado segundo as faixas de IMC (abaixo do peso, normal, sobrepeso, obesidade).
- Exibição dinâmica do resultado na tela.
- Link adicional para que o usuário possa acessar mais informações sobre o IMC após o cálculo.

Ferramentas utilizadas:
HTML, CSS e JavaScript puro.

Previsão do Tempo

Previsão do Tempo

Previsão do Tempo

O projeto "Previsão do Tempo" é uma aplicação web desenvolvida em React que permite ao usuário consultar a previsão do tempo em tempo real. Através de uma interface simples e intuitiva, o usuário pode buscar as condições climáticas para uma cidade específica.

Principais funcionalidades:
- Exibição da previsão do tempo atual para uma cidade inserida pelo usuário.
- Informações detalhadas, como temperatura, umidade e condições climáticas.
- Atualização dinâmica das informações com base na cidade fornecida.

Ferramentas utilizadas:
- React: Framework utilizado para a construção da interface e gerenciamento do estado da aplicação.
- Axios: Biblioteca para realizar requisições HTTP à API de previsão do tempo.
- Vite: Ferramenta de build e desenvolvimento rápido para aplicações React.
- gh-pages: Para o deploy da aplicação no GitHub.

E-commerce

E-commerce

Frontend Mentor - Página de Produtos de E-commerce

Este projeto é uma solução para o desafio de criação de uma página de produtos de e-commerce proposto pela plataforma Frontend Mentor. O objetivo do desafio é desenvolver uma interface funcional e responsiva baseada em um layout fornecido, replicando recursos comuns de um site de loja virtual.

O Desafio: - Visualizar o layout ideal para diferentes tamanhos de tela (responsivo)
- Ver os estados de foco em todos os elementos interativos.
- Abrir uma galeria lightbox ao clicar na imagem principal do produto.
- Trocar a imagem principal clicando nas miniaturas.
- Adicionar itens ao carrinho
- Adicionar itens ao carrinho

Ferramentas utilizadas:
- HTML5 semântico: Utilizado para estruturar o conteúdo de forma acessível.
- CSS personalizado com variáveis: Para manter um design consistente e reutilizável.
- Flexbox e Grid Layout: Para controle de layout moderno, flexível e responsivo.
- Media Queries: Para adaptar o design a diferentes tamanhos de tela.
- JavaScript puro (Vanilla JS): Responsável pela interatividade como troca de imagens, carrinho de compras e lightbox.

Tela Login

Mini Projeto - Tela de Login

Este é o primeiro projeto que desenvolvi sozinho, com o objetivo de treinar estruturação e estilização de uma interface usando apenas HTML e CSS. Trata-se de uma tela de login simples, com foco em design limpo, alinhamento de elementos e responsividade básica.

Mini Projeto - Tela de Celular com Redes Sociais

Este projeto foi desenvolvido como parte de um desafio proposto no curso de HTML5 e CSS3 do Curso em Vídeo, ministrado pelo professor Gustavo Guanabara. O objetivo era criar uma interface simulando a tela de um celular, com links para redes sociais.

- Imagem do perfil na tela

- Layout simples, limpo e agradável
- Responsividade básica para se adaptar bem em diferentes tamanhos de tela

Ferramentas utilizadas:
- HTML5 semântico: Para estruturar os elementos da tela de forma clara e acessível.
- CSS personalizado: Para todo o estilo visual da interface, incluindo cores, ícones, botões e layout.
- Flexbox: Utilizado para organizar os elementos verticalmente e centralizá-los na tela.
